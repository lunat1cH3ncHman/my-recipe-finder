# my-recipe-finder

MERN stack - Mongo, Express, React, Node

Passport and JWT for auth

Create a local .env file with:

MONGO_ROUTE=mongodb://

PASSPORT_SECRET=

Launch backend and front end locally:

- npm start dev

Deploy to Heroku:

- git push heroku master

Lauch backend only:

- npm start

Launch frontend only:

- cd client
- npm start
